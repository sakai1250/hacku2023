## kikonAshI
Kikonashiは、「HackU名城大学2023」にてビビっとが独自開発したファッションAIを搭載する、iOSアプリです。3つの機能で悩める皆さんをサポートします。

<img src="https://github.com/sakai1250/hacku2023/assets/92532910/e296ff7c-7809-42c7-92cf-9b95e1365334" width="50%">

１,おしゃれなコーディネート提案！
自分が持っている服を写真に撮ってアプリに登録すると、季節や天気、あなたの好みに合わせて、AIがおしゃれなコーディネートを提案してくれます。気に入らなければ断ってしまえばok。皆さんの好みに合わせてAIが学習していきます。
次に，残りの2つの機能を紹介します．

２,コーディネートのおしゃれ度測定！
上下の服の組み合わせを選択すると、コーディネート投稿サイトの人気ランキングを元に、AIがおしゃれ度を測定してくれます。

３,服の購入をサポート！
気になった服の画像をアプリに登録すると、自分が持っているどの服とマッチするか、AIが教えてくれます。

スマホで快適に動作する軽量なAIモデルを独自開発しました。あなたのスマホ内で完結して動作するので、プライバシーも安心です。また、独自開発したAIは、季節、天候、男女別に服の提案ができるよう、16種類用意しています。
<img src="https://github.com/sakai1250/hacku2023/assets/92532910/f6711ec6-0bec-4008-9080-edb863f21448" width="50%">

「診断」
主要機能の、コーディネート提案を試します。服は事前に登録してあります。

「推論」
登録した上下の服の、すべての組み合わせに対して、おしゃれ度を測定しています。

「おしゃれ度表示」
おしゃれ度が一番高いコーディネートが提案されます。
気に入らなかったら、やめとく、を押してみましょう。
次におしゃれ度が高かったコーディネートを提案してくれます。
これにする、を押すと、今までの回答をもとに、AIがユーザーに合わせて再学習します。
「再学習」
あなたと一緒にキャラクターもおしゃれになっていきます。

## Privacy Policy
利用者情報の取得、利用
本アプリでは、天気、、季節を取得するために端末の位置情報を取得する場合があります。

利用者情報の第三者提供
アプリが利用者の個人情報を第三者へ提供する場合は、利用者の個人情報の提供先を書いたほうが良いです。

お問い合わせ先
メールアドレスや電話番号などの連絡先を書いた方が良いです。アプリ開発者に問い合わせする手段が無いとダメらしいのでお問い合わせ先は書いておきましょう。
